# Political Analysis Daily Report

时政分析日报生成器 —— 基于实时新闻数据，以学术国际关系理论框架分析国内外时政，输出含图表的 docx 报告。

## 功能概览

- **国内时政** — 政策法规动态、人事调动追踪、重要会议与讲话、经济政策信号、社会治理、军事国防
- **国际时政** — 大国博弈、地缘冲突、多边关系、全球经济治理、区域热点
- **学术框架分析** — 现实主义 / 自由主义 / 建构主义 IR 理论驱动的形势研判
- **前瞻预测** — 国内政策窗口期判断 + 国际战略态势预测（附时间维度）
- **图表可视化** — 新闻分类分布、区域热度、大国关注度对比、时间线趋势
- **真实数据** — 新浪新闻公开 API，curl 直连，无需注册

## 快速开始

```bash
# 安装依赖
pip3 install python-docx matplotlib numpy

# 运行
python3 generate_political_report.py
```

输出路径：`~/Desktop/项目/时政报告/时政分析日报_YYYY-MM-DD.docx`

## 分析框架

报告基于以下国际关系学术理论构建分析视角：

- **现实主义 (Realism)** — 大国权力竞争、安全困境、势力均衡
- **自由主义 (Liberalism)** — 国际制度合作、经济相互依存、民主和平
- **建构主义 (Constructivism)** — 国际规范演化、身份认同、话语塑造
- **议程设定理论 (Agenda-Setting)** — 国内政策窗口期分析
